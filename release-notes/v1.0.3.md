# Relax Block Puzzle Android v1.0.3

This is an Android debug/preview build.

## Update

- Improved drag experience: when picking up a block from the rack, it smoothly lifts from its original position to above the finger with a slight scale-up animation.
- The dragged block is now hidden from the rack during drag, eliminating the "two blocks" visual issue.
- Increased finger offset so the floating block preview is further from the finger, reducing finger occlusion.
- Removed legacy shooting game code and unused audio files (cleanup).

## Stability Kept From v1.0.2

- Stable single-canvas Android rendering path
- Android canvas DPR cap
- Dirty render optimization
- Background lifecycle optimization
- Normal Chinese text in settings
- Stable home layout

## Core Features

- 10x10 block puzzle board
- Three difficulty levels
- Local best scores
- Power-ups: refresh, clear, undo
- Local welfare code
- Offline Android play

This is an independent casual block puzzle project and is not affiliated with any official game brand.

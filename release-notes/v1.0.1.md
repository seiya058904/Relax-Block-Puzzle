# Relax Block Puzzle Android v1.0.1

This is an Android debug/preview build.

This is the current confirmed stable Android APK version built from the latest repository source.

## Fixes and Stability Updates

- Fixed Android drag responsiveness and improved touch feel when moving blocks.
- Fixed background transition jank when entering the background or recent apps.
- Stabilized the single-canvas Android rendering path.
- Limited Android canvas DPR for better rendering performance and smoother app switching.
- Optimized Android background lifecycle handling for safer pause/resume behavior.
- Confirmed settings page Chinese text displays correctly.
- Confirmed the home layout remains stable and readable.

## Core Features

- 10x10 block puzzle board
- Three difficulty levels
- Local best scores
- Power-ups: refresh, clear, undo
- Local welfare code
- Offline Android play

## Notes

- This build is intended as an Android debug/preview APK.
- The APK is kept out of the source repository and should be distributed through GitHub Releases.

# Relax Block Puzzle Android v1.0.2

This is an Android debug/preview build.

This is a small Android experience update based on the confirmed stable v1.0.1 build.

## Update

- Slightly increased the drag offset so the dragged block appears a bit higher above the finger.
- Keeps the v1.0.1 rendering stability and background transition fixes.

## Stability Kept From v1.0.1

- Stable single-canvas Android rendering path
- Android canvas DPR cap
- Dirty render optimization
- Background lifecycle optimization
- Normal Chinese text in settings
- Stable home layout

## Core Features

- 10x10 block puzzle board
- Three difficulty levels
- Local best scores
- Power-ups: refresh, clear, undo
- Local welfare code
- Offline Android play

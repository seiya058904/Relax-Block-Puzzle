# Relax Block Puzzle Android v0.1.0 Preview

This is a preview/debug Android build.

Built from the current repository source.

APK file: `Relax-Block-Puzzle-Android-v0.1.0-preview.apk`

Core features:

- 10x10 block puzzle board
- Three difficulty levels
- Local best scores
- Power-ups
- Local welfare code
- Offline Android play
